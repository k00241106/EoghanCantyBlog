Entry 3 for October

Eoghan Canty Fri 5/11, 2021

For my final entry of the month I want to talk about how I finished the research report. Starting the technologies section I first wanted to identify which ones I would be covering in this report. I broke it down into each of the sections of my project. So, for example how would I build the backend architecture of my website? How would I build the front-end? Breaking it down into almost an agile based system has allowed me to identify each of the parts I’ve needed. The first technology was Augmented Reality. As a big part of my project is based around AR I wanted to cover this technology. I wanted to explain the importance of AR in education but also how this technology is being underused. Another technology I am using in this project is 3D technology. In my AR app all of the models of F1 cars will be 3D. Therefore, I must build these out using software such as Blender to immerse the user into an actual real world F1 environment. To build out my website I will be using a very popular web development stack. This is the MERN stack. The MERN stack is made up of MongoDB, Express, React and Node JS. MongoDB will act as my database. Express will allow me to use a framework for node JS. React will be my front-end framework and finally Node JS will allow my JavaScript to run in this backend environment. Unity 3D will allow me to create and run my AR experience. Having used this in the past for a previous AR project this was an easy choice. To write my code I will use Visual Studio Code for my website and Visual Studio for my AR app. Finally, to edit the videos I will be making, I will need a top class editing software. Adobe Premier Pro is without doubt the first choice due to its simplicity and my past experience using it. Moving onto the design concepts section I had to research the user experience here. Understanding the importance of responsive design is key after some of the research I did. Also, the AR app experience would need to be as simple as possible. As AR apps are very underused there isn’t common style techniques. The next section, immersion was a study in how users want to feel immersed whether it’s AR or VR. When we apply real world technologies like these users want to feel like they are part of this world. So, having bugs or graphical glitches will completely take way the immersion. The final section in design concepts is search engine optimisation. In this section of report, I built out a keyword universe. As my project is about building an educational brand I want to rank high in Google search rankings. The penultimate section of the report I did was the deliverables. This section was simply all of the products I wanted to submit come the end of the project. Finally, the last section of the research report was simply my conclusion from the report.

Overall, this month was a difficult month due to time constraints forced by other classes but in the end I managed to upload a report that I was delighted with in the end. I felt like I researched all the relevant topics that would help my project excel.
